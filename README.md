Этот репозиторий мое портфолио который содержит следующие проекты:


java-filmorate-main - 
бэкэнд сервиса  который помогает пользователям выбтрать фильм для просмотра на основе оценок других пользователей, по отзывам или по общим с друзьями фильмам. 
Позволяет искать фильмы по названиям ,описаниям, жанрам, годам, режисерам, рейтингу, оставлять комментарии к фильмам, показывать ленту событий друзей,
а так же может рекомендовать фильмы к просмотру с помощью алоритма рекомендаций.
использованы - Maven, Spring Boot в соответствии с REST. Для работы с БД PostgreSQL и Н2 использовался Spring JDBC, для логирования использован SLF4J,

java-shareit-main - 
бэкэнд многомодульного Maven приложения для шеринга вещей. Основной сервис позволяет пользователям размещать вещи которые они готовы сдать в аренду ,
а так же находить нужные им вещи бронировать их после чего брать их в аренду и оставлять отзывы. Дополнительный сервис это gateway
в который вынесена валидация входящих данных. Т.к реализована микросервисная архитектура значит приложение легче поддерживать и масштабировать
использованы - REST, Spring Boot, Spring Data JPA, Hibernate, PostgreSQL, Docker, docker-compose. для тестов Mockito ,для логирования использован SLF4J

java-explore-with-me-main - бэкэнд микросервисного Maven приложения - афиши где пользователи могут предложить какое-либо событие 
от выставки до похода в кино и собрать компанию для участия в нём. Основной сервис позволяет создавать события, находить события используя разные фильтры, 
подавать заявки на участие в событиях, администрировать события и пользователей. Сервис статистики хранит информацию о количестве обращений пользователей
к спискам событий и о количестве запросов к подробной информации о событии. Эта статистика используется в основном сервисе для сортировки по популярности,
а так же на основе этой информации должна формироваться статистика о работе приложения.
использованы  Maven, REST, Spring Boot, RestClien, Spring Data, Hibernate, PostgreSQL, Docker, docker-compose, Postman, для логирования использован SLF4J
